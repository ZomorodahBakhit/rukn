The file contains:

- 6 Homepage variants (Maghrib base + Sounds-open / Timer-open / Scenes-open + Olive + Library)
- 02 Login
- 03 Dashboard (with floating glass widgets, dock, Try Premium pill)
- 04 Library (with 7 unique illustrated book covers, working Filter + Sort dropdowns)
- 05 Reader
- 06 Journal
- 07 Settings (Account / Preferences / Plan / Account data — 4 sub-variants)
- 08 Premium upgrade screen
- Modal: Delete account confirm
- Overlays: Library Filter dropdown, Library Sort dropdown

Every screen is built natively in Figma — no image-fill placeholders. All buttons are wired with click→navigate / click→open overlay reactions. The Scene window's three thumbnails actually switch the homepage scene (Maghrib / Olive / Library) — each is a separate frame variant with its own sky / sun / hill colors. The Sound, Timer, and Scene dock buttons swap to a sibling frame where both the matching window appears AND the button gains its active amber highlight; the × inside the window navigates back to the base, which clears the highlight.

All icons are real custom-drawn vector strokes (no emojis anywhere in the UI chrome).

## How to test the prototype

1. Open the **Play prototype** link above.
2. Press the spacebar or click any button.
3. Use the bottom-left "Restart" button if you get lost.
